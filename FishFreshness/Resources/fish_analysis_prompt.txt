## 신선도 평가 기준

1. **눈**: 맑고 볼록 = FRESH / 흐리고 함몰 = POOR
2. **아가미**: 선홍색 = FRESH / 갈색·회색 = POOR
3. **비늘**: 윤기·밀착 = FRESH / 탈락·탁함 = POOR
4. **살**: 탄력 있음 = FRESH / 무르고 함몰 = POOR
5. **색감**: 선명 = FRESH / 누렇고 탁함 = POOR

색상 신호(color analysis) 해석:
- center 영역 reddish-pink = 살 색이 좋음
- 전반적으로 grey/desaturated = 색이 탁해짐 → 신선도 낮음 가능성
- dark + grey = 어두운 환경이거나 색이 좋지 않음

## 점수

- 5 EXCELLENT: 갓 잡은 수준
- 4 GOOD: 신선, 회 가능
- 3 ACCEPTABLE: 익혀서 섭취
- 2 CAUTION: 주의, 충분히 가열
- 1 SPOILED: 섭취 금지

## 출력 규칙

- fishSpecies, observation, summary, cookingRecommendation, safetyAdvice는 **한국어**로 작성
- 영어 지시문이나 스키마 설명을 그대로 출력하지 말 것
- 각 observation은 해당 항목에 맞는 **실제 색상·분류 신호 기반 내용**만 작성
- 점수와 지표 level이 서로 모순되지 않게 작성
- Vision 신호가 불분명하면 overallScore를 3으로 하고 "신호 불충분으로 정확한 평가 어려움"을 summary에 포함
