# FishFreshness

FishFreshness는 생선 사진 한 장으로 신선도 점수, 등급, 조리·안전 조언을 제공하는 iOS 앱입니다. Apple Intelligence(Foundation Models)를 on-device로 사용하며, 분석 기록은 SwiftData로 저장합니다.

## Requirements

- iOS 26.0+
- Xcode 26+
- Apple Intelligence 지원 기기 (분석 기능)

## Setup

자세한 Xcode 설정은 [FishFreshness/XCODE_SETUP.md](FishFreshness/XCODE_SETUP.md)를 참고하세요.
